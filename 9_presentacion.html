Sistema Operativo

Teoría fundamental para estudiantes de Informática

El software principal de cualquier dispositivo electrónico.
Funciona como puente entre el usuario y la máquina.

¿Qué es el Sistema Operativo?

Software principal que administra el hardware y permite que otras aplicaciones funcionen.

🧠 Gestión de recursos

  • Asigna memoria RAM
  • Distribuye tiempo de procesador (CPU)
  • Administra espacio de almacenamiento

🖥️ Interfaz de usuario

  • Entorno visual (escritorio, ventanas)
  • Interfaz de comandos (CLI)
  • Permite la interacción humano-máquina

📁 Administración de archivos

  • Organiza la información
  • Guarda y recupera datos
  • Gestiona permisos de acceso

🔌 Control de dispositivos

  • Teclado y mouse
  • Impresoras y escáneres
  • Comunicación hardware-software

¿Qué es el Kernel?

El núcleo de un sistema operativo.

Aplicaciones
Software de usuario
↕
KERNEL

Hardware
CPU, RAM, Disco

⚡ Funciones del Kernel

  • Administrar memoria RAM
  • Controlar la CPU
  • Gestionar discos
  • Administrar dispositivos
  • Comunicar software y hardware

🚨 Dato importante

Linux NO es un sistema operativo completo.

Linux es un kernel.

Ubuntu, Debian y Fedora sí son sistemas operativos completos que utilizan el kernel Linux.

Historia de los Sistemas Operativos

Los primeros pasos de la computación moderna

1956 – GM-NAA I/O

  • Primer sistema operativo conocido
  • Desarrollado para la IBM 704
  • Automatizaba procesos repetitivos

1962 – Desarrollo de UNIX

  • Bell Labs inicia investigaciones
  • Sistemas operativos avanzados

1969 – Nace UNIX

  • Creado por Ken Thompson y Dennis Ritchie
  • Multiusuario y multitarea
  • Sistema de archivos jerárquico
  • Seguridad mediante permisos

1971 – UNIX Versión 1

  • Primera versión oficial

1975 – Microsoft BASIC

  • Microsoft publica BASIC

1981 – MS-DOS

  • Sistema basado en comandos
  • Dominó las PC durante los años 80

1983 – Apple Lisa

  • Primer sistema comercial con interfaz gráfica avanzada

1984 – Macintosh

  • Populariza ventanas, iconos y ratón

1985 – Windows 1.0

  • Primera interfaz gráfica de Microsoft

Historia de los Sistemas Operativos (II)

La era de las distribuciones Linux y Windows moderno

1987 – OS/2

  • Desarrollado por IBM

1992 – Slackware

  • Una de las primeras distribuciones Linux

1992 – MCC Interim Linux

  • Primera distribución Linux pública

1993 – Debian

  • Proyecto comunitario
  • Estabilidad y software libre
  • Base para muchas distribuciones

1993 – Red Hat Linux

  • Se convertiría en la distribución empresarial más importante

1994 – SuSE Linux

  • Creada en Alemania

1995 – Windows 95

  • Menú Inicio
  • Barra de tareas
  • Plug and Play

1996 – Windows NT 4.0

  • Orientado a empresas y servidores

1998 – Windows 98

  • Soporte USB mejorado
  • Plug and Play avanzado
  • Soporte para DVD

2000 – Windows 2000

  • Diseñado para ambientes empresariales

2001 – Windows XP

  • Uno de los sistemas operativos más exitosos

2001 – Mac OS X

  • Basado en UNIX

Historia de los Sistemas Operativos (III)

Consolidación de Linux, móviles y la nube

2002 – RHEL

  • Red Hat Enterprise Linux para el mercado empresarial

2003 – Fedora

  • Proyecto comunitario patrocinado por Red Hat

2004 – Ubuntu 4.10

  • Lanzado por Canonical
  • Fácil instalación, gratuito
  • Amplia comunidad

2005 – openSUSE

  • Versión comunitaria de SUSE

2007 – Windows Vista

  • Interfaz Aero
  • Mejoras de seguridad

2007 – iOS

  • Aparece junto al primer iPhone

2008 – Android

  • Presentado por Google, basado en Linux

2009 – Windows 7

  • Reconocido por su estabilidad

2011 – ChromeOS

  • Llega al mercado

2012 – Windows 8

  • Introduce la interfaz Metro

2014 – Linux Mint

  • Se consolida como distribución popular para hogar

2015 – Windows 10

  • Microsoft adopta actualizaciones continuas

Historia de los Sistemas Operativos (IV)

De la popularización de Linux hasta la era de la IA

2016 – Manjaro

  • Acerca la potencia de Arch Linux a usuarios menos experimentados

2018 – Elementary OS

  • Destaca por su diseño inspirado en macOS

2019 – Linux domina la nube

  • Principales plataformas de computación en la nube usan Linux

2021-2022 – Windows 11

  • Anunciado en 2021, lanzado oficialmente en 2022

2023 – Debian 12 Bookworm

  • Nueva versión estable de Debian

2024 – Ubuntu 24.04 LTS

  • Una de las versiones LTS más utilizadas
  • También destacan: Fedora 40, Linux Mint 22, openSUSE Leap, Arch Linux

2025 – Consolidación de la IA en los SO

  • Inteligencia artificial integrada
  • Automatización avanzada
  • Computación en la nube nativa

2026 – Situación Actual

  • Distribuciones Linux consolidadas: Ubuntu, Debian, Fedora, RHEL, Linux Mint, Arch, Manjaro, openSUSE, Elementary OS
  • SO dominantes: Windows 11, Android, iOS, macOS, Linux
